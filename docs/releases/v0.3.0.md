# szn v0.3.0

**6 commits** since v0.2.0.

---

## Headline: Precise Thai Reflow & Manual Reflow Control

This release completes the pane reflow capabilities introduced in v0.2.0 by integrating dictionary-based Thai word-wrapping and manual reflowing commands:

- **Dynamic `libthai` Integration** — Dynamically loads the `libthai` library on Unix environments (`libthai.dylib` / `libthai.so`) to provide dictionary-level syllable and word wrapping.
- **Robust Library Paths** — Fallback searches extended to 64-bit Linux search paths (`/usr/lib64`, `/usr/local/lib64`), ensuring seamless compatibility across different architectures.
- **Manual Pane Reflow Command** — Adds the `reflow-pane` command and maps it to the `Ctrl-b r` keyboard shortcut.
- **Thai-Bordering Space Wrapping** — Solves wrapping bugs when Thai words border trailing spaces.

---

## New Features

- **`reflow-pane` command** — Manually triggers reflow logic on a pane.
- **`Ctrl-b r` key binding** — Standard shortcut mapping to execute the `reflow-pane` command.
- **Dictionary-based Word Boundary Lookup** — Seamless loading and integration of `libthai`'s `th_brk_wc_find_breaks` API.
