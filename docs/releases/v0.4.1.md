# szn v0.4.1

**5 commits** since v0.4.0.

---

## Headline: Refined Mouse Selection, Hover Focus Scrolling, and Event Loop Crash Fix

This patch release delivers important bug fixes and refinements to the mouse drag selection, pane scrolling, and event loop logic introduced in v0.4.0:

- **Mouse Drag Selection Improvements** — Fixes coordinate drift when auto-scrolling at pane boundaries and resolves a highlighting alignment issue when scrolling the pane manually. Dynamic translation is now used to keep selection highlights correctly anchored to their underlying grid lines.
- **Hover Pane Focusing and Scrolling** — Hovering over any pane and using the mouse scroll wheel now automatically focuses that pane. If the running program does not capture mouse input, scrolling automatically scrolls history back/forward (entering or exiting copy-mode as needed).
- **Edge Auto-Scrolling** — Drag-selecting text to the top or bottom boundaries of a pane now triggers timer-driven continuous auto-scrolling to ease selecting larger blocks of scrollback.
- **OSC Buffer Crash Fix** — Resolves a critical server crash (Bug #138) caused by resetting state on an empty OSC sequence buffer.
