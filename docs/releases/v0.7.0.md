## Headline: Status Bar, Pane Borders, and a 39-Bug Cleanup Sweep

v0.7.0 delivers the long-awaited tmux-compatible status bar with a rewritten
format engine, pane-border-format indicators, and a systematic 39-bug cleanup
covering everything from scroll history corruption to sixel eviction,
double-frees, and arena allocation discipline.

- **Status Bar** â€” Full tmux-compatible status bar driven by `status-left`,
  `status-right`, `status-justify`, `window-status-format`, and
  `window-status-current-format` options. Supports `status off` to hide the
  bar and `status-interval` for periodic clock updates. The format engine was
  rewritten with `TemplateCache` (compile-once), stride-free deduplication,
  `#[style]` sequences, `strftime`, and single-character aliases (`#S`, `#I`,
  `#W`, `#P`, `#H`, `#h`, `#F`, `#D`, `#T`) matching tmux conventions.
- **Pane Border Format** â€” `pane-border-format` (default `#P` for pane index)
  draws on the top border of each pane in vertical splits, and vertically
  down the left border in horizontal (side-by-side) splits. Both horizontal
  and vertical-border rendering are supported.
- **Systematic Bug Sweep (#210â€“#248)** â€” 39 bugs found through deep code
  audits, static analysis, and manual review. Every one is fixed. The bug
  tracker now holds 248 entries, all with uniform `Status:` metadata.
