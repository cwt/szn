## Headline: Text Reflow

szn now rewraps text when a pane is resized, preserving word/line integrity
and scrollback history. Includes:

- **Thai cluster detection** ‚Äî clusters split across line boundaries on resize
  are re-joined; individual clusters are treated as indivisible wrapping units
- **Thai syllable look-ahead** ‚Äî $O(1)$ backtracking heuristic prevents
  orphan consonants from being split from their leading vowels
- **Number wrap** ‚Äî short numbers ($\le$ 6 chars) are kept intact; long
  numbers break after commas
- **CJK double-width integrity** ‚Äî wide characters and their padding cells
  stay together
- **Lossless drag-resize** ‚Äî distinguishing padding cells (`char = 0`) from
  typed spaces (`char = ' '`) prevents text degradation across repeated
  resizes
- Cursor position tracking during reflow prevents prompt doubling
- Removed 80√ó24 minimum clamp on terminal and client resize paths

See [`docs/TEXT_REFLOW.md`](../TEXT_REFLOW.md) for the full design.

## New Features

- **Paste buffers** ‚Äî `set-buffer`, `paste-buffer`, `list-buffers`,
  `choose-buffer` with interactive menu
- **Clock mode** ‚Äî `clock-mode` command with `-u` flag for UTC; local
  timezone via `localtime_r()`
- **Command prompt** ‚Äî tab-completion and interactive selection menu
- **Copy-mode visual indicators** ‚Äî yellow border and status bar highlight
- **`send-prefix`** action with default `C-b` binding to forward a literal
  `C-b` to the shell
- **Nested szn guard** ‚Äî detects and warns when attaching from inside another
  szn instance
